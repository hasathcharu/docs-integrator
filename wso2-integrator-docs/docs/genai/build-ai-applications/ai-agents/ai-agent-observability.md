# AI Agent Observability

Observability is critical for understanding how your AI agents behave in production. This guide covers how to monitor, trace, and debug agent execution.

## What Agent Observability Covers

- **Agent Tracing** -- End-to-end tracing of the agent loop: user input, LLM calls, tool invocations, and decision points.
- **Tool Call Monitoring** -- Tracking which tools are called, how often, and how long they take.
- **LLM Interaction Logging** -- Capturing prompts sent to the model and responses received.
- **Memory Usage Tracking** -- Monitoring how memory is being used across conversation turns.
